# Player Debuffs

A lightweight World of Warcraft addon for The Burning Crusade that mirrors the player's debuffs below their unitframe.

## Features

-   Mirrors your debuffs in a movable frame.
-   Lightweight and efficient.
-   In-game configuration panel.

## How to Use

Type `/pd` in the chat to open the settings panel.

## Options

The settings panel allows you to customize the following:

-   **Icon Scale:** Adjust the size of the debuff icons using a slider.
-   **Debuff Priority:** Set the priority of different debuff types (Magic, Curse, Disease, Poison) using a slider. Debuffs with a lower priority number will be displayed first.

## Dependency

This addon requires the [Ace3](https://www.wowace.com/projects/ace3) library to be installed.
